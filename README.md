# Super Store Sales Analysis Project 📊

## 📝 Project Overview
This project involves a comprehensive sales data analysis for a retail store. The goal was to transform raw data into an interactive and insightful dashboard using Advanced Excel tools.

## 🛠️ Data Engineering Steps
1. **Data Cleaning & ETL:** Imported raw data using **Power Query**, performed data cleaning, and handled missing values.
2. **Data Normalization:** Normalized the dataset by splitting it into structured tables:
   * **Orders** (Fact Table)
   * **Customers, Products, Regions** (Dimension Tables)
3. **Data Modeling:** Built a **Relational Data Model** in **Power Pivot**, establishing one-to-many relationships between tables.
4. **DAX & Calculations:** Created **Calculated Measures** and **Calculated Columns** to track business performance.

## 📈 Dashboard Features (KPIs)
The interactive dashboard provides insights into:
* **Total Sales & Profitability**
* **Top Performing Products**
* **Regional Performance Analysis**
* **Customer Segmentation**
